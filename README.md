# chill-assistant

**過集中しないための、静かな相棒。**

作業中やゲーム中に、集中を邪魔せず、壊れないペースを守るためのデスクトップ常駐アプリです。



## これは何？

chill-assistant は、
**「集中できる人が、集中しすぎて消耗しないため」**の寄り添いアプリです。

生産性を最大化するツールでも、
ただ癒やすマスコットでもありません。

* 集中は続けたい
* でも止め時がわからなくなる
* 気づいたら疲れ切っている

そんな状態を前提に、
**そばにいて、空気を壊さず、必要なときだけ合図する**
──そのための「環境寄りアシスタント」を目指しています。



## どんな立ち位置？

* Desktop Mate のような「見るだけの癒やし」とは違う
* ガチガチの作業効率化ツールとも違う
* *chill 作業お供 / ゲームのお供* に近い距離感

主役はユーザーで、
chill-assistant は **同席者** です。

話しすぎない
指示しない
急かさない
でも、完全に無言でもない

この「中途半端さ」を、意図的に選んでいます。



## 想定しているユーザー

* 過集中しがちな人
* ゲームや作業に没頭しやすい人
* 集中力はあるが、燃え尽きやすい人

ただし「過集中の当事者専用」ではありません。

誰でも

* 夜ふかしする日
* やめ時を逃す日
* 一人で黙々とやる日
  はあります。

chill-assistant は
**過集中“しやすい瞬間”に効く**ことを重視しています。



## 設計の思想

### 1. 集中を邪魔しない

* 通知は最小限
* 介入はやさしく、短く
* 罪悪感を煽らない

「止める」より「気づかせる」。

### 2. アプリとして自立する

LLM や Voice が発展しても、
それらに依存しない核を持ちます。

* ローカルで成立する観測（時間・操作・状態）
* ルールベースでも機能する介入
* モデルは **強化パーツ** として任意

最悪、AIが使えなくても成立する。
それが長く育てられる条件だと考えています。

### 3. 開発も過集中しない

このプロダクト自体も、
「無理なく触り続けられる」ことを重視します。

完璧を目指さず、
育てる余白を残す。



## 現在のステータス

## やらないこと（少なくとも今は）

* 常時会話・過剰なキャラクター表現
* ユーザーを管理・評価する設計



## 名前について

**chill-assistant**
chill を主に、assistant を従に。

癒やしすぎず、役に立ちすぎない。
その中間を目指しています。
